API

The API Documented here is an early version of the api. Methods are subject to change.

VApi

The VApi global object is available from VApi window.VApi and global.VApi.

Meta

The Meta object is the first available property in VApi

it contains information about the Velocity Build, Discord Version and VApi Version

Properties

React

Discord's React Module

ReactDOM

Discord's ReactDOM Module

Methods

request()

A modified version of OpenAsar's request Polyfill

showChangelog()

Shows the changelog for the most recent version of Velocity

showInfoModal()

Shows an info modal with information about your Velocity installation.

showToast(title, content, options = {})

Shows a toast message

Options

const { type = "", timeout = 3000, color = "" } = options

Name Description Type Default Values
type Style of toast string "" "error", "warn", "success"
timeout [ms] before toast closes int 3000
color Toast Color string ""

showConfirmationModal(title, content, options = {})

Shows a confirmation modal with the given title, content and options

Options

Name Description Type Default Values
onConfirm Function to execute on confirm function ()
onCancel Function to execute on cancel function ()
confirmText Label on Confirm button string "Okay"
cancelText Label on Cancel button string "Cancel"
danger Whether to display the action as dangerous boolean false true, false
key modalKey any undefined

Modules

Logger

Logger Module

Logger.log(title, ...logs)

Logs a message with the given title and logs

Logger.warn(title, ...warnings)

Logs a warning with the given title and warnings

Logger.error(title, ...errors)

Logs an error with the given title and errors

WebpackModules

Get and interact with Discord's internal Modules

WebpackModules.find(query)

Shorthand method for finding a module by the given query

VApi.WebpackModules.find("UserStore")

  • Will find a module with the displayName of "UserStore"

VApi.WebpackModules.find(["UserStore"])

  • Will find a module with props titled "UserStore"

VApi.WebpackModules.find((m) => m.default?.displayName === "UserStore"])

  • Will filter modules with a displayName of "UserStore"

WebpackModules.findByDisplayName(query)

Method for finding a module by the given displayName

WebpackModules.findByDisplayNameDefault(query)

Equivalent to WebpackModules.findByDisplayName(query).default

WebpackModules.findByProps([queries])

Method for finding a module by the given props

WebpackModules.findByPropsDefault([queries])

Equivalent to WebpackModules.findByProps([queries]).default

Utilities

Random Utilities

Utilities.joinServer(code, goTo)

Joins a server with the given invite code (if goTo is true, will move to the server)

VApi.Utilities.joinServer("123456789", true)

Utilities.joinOfficialServer()

Joins the official Velocity server.

VelocityElements

Velocity's custom DOM elements

VelocityElements.head

The velocity-head DOM Node

VelocityElements.body

The velocity-body DOM Node

Styling

Styling Module

Styling.injectCSS(id, css)

Injects CSS into the DOM

Styling.clearCSS(id)

Clears CSS from the DOM

Styling.linkStyle(id, url)

Adds a reference to an external Stylesheet

Styling.removeStyle(id)

Removes a reference to an external Stylesheet

Scripting

Scripting Module

Scripting.appendScript(id, url)

Adds a reference to an external script to the DOM

Scripting.removeScript(id)

Removes a reference to an external script from the DOM

DataStore

DataStore Module

DataStore.getData(PluginName, key)

Gets data from the DataStore

DataStore.getAllData(PluginName)

Gets all data from the DataStore for a specific Plugin

DataStore.setData(PluginName, key, value)

Sets data in the DataStore

DataStore.deleteData(PluginName, key)

Removes data from the DataStore
