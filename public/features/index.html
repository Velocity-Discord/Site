Themes

Our Theme system allow you to customise the look of your Discord however you want, just using plain CSS. Velocity lets you import styles from almost everywhere! Our Custom @var and @if allow you extend Native CSS!

Plugins

Plugins can make use of VApi and Discord's internal functions to extend what Discord can do and add functionality.

                    
const { React, showToast } = VApi 

React.createElement("button", {
    className: "my-cool-button",
    style: {width: "20px"},
    onClick: () => {
        showToast("Clicked", {type: "info"})
    },
}, "text!")
